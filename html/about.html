关于 PureCpp

中国最纯粹的 C++ 技术社区，从2015年到2026年，从 C++11 到 C++26，只聊硬核技术，不灌水，不广告。

社区简介

purecpp.cn 是一个致力于推动现代C++ 最新技术在中国企业应用的专业社区。自2015年成立以来，我们一直坚持"纯粹技术，深度交流"的理念，为广大 C++ 开发者提供了一个高质量的学习和交流平台。

自2018 年以来每年都举办purecpp 大会，聚集最新C++标准、最佳实践、开源项目等方面，为开发者提供最新的技术资讯和交流平台。

我们的社区聚集了来自全国各地的 C++ 技术爱好者、从业者和专家，涵盖了从初级开发者到资深架构师的各个层次。在这里，你可以讨论 C++ 语言的最新特性、分享项目经验、解决技术难题、参与开源项目，以及了解行业动态。

我们拒绝灌水和广告，专注于技术内容的深度和质量。通过严格的内容审核机制和积极的社区管理，我们努力营造一个纯粹、专业、友好的技术交流环境。

使命与愿景

使命

为中国的 C++ 开发者提供一个纯粹、专业的技术交流平台，促进现代 C++ 技术的普及和发展，推动中国软件行业的技术进步。

愿景

成为中国最具影响力的 C++ 技术社区，连接全球 C++ 开发者，共同推动 C++ 生态系统的繁荣和发展。

价值观

  • 纯粹技术：专注于技术内容的深度和质量，拒绝灌水和广告
  • 开放包容：欢迎不同层次、不同背景的开发者，鼓励多样化的技术观点
  • 互助共享：倡导知识共享和互助精神，共同成长
  • 持续学习：紧跟技术发展趋势，不断学习和进步

发展历程

2015年

purecpp 社区正式成立，开始聚集 C++ 技术爱好者

2017年

社区网站改版，后端全部改成C++ 实现

2018年

举办第一届 PureCpp 技术大会，邀请国内外知名专家演讲，每年都会举办

2026年

新版purecpp 网站上线，正成为国内最具影响力的 C++ 社区之一
